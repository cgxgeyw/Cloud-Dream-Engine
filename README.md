# 云朵梦境 Cloud Dream Engine

云朵梦境是一套以“世界包”为核心的 AI 叙事游戏创作与体验引擎。它把世界设定、角色、地图、提示词、状态面板、游戏内 UI、素材资源和存档调试能力组织在一起，让一个 AI 剧情世界不只是一段聊天，而是一套可以被创作、游玩、迁移和持续迭代的完整体验。✨

## 这是什么

云朵梦境面向想要制作 AI 剧情、角色扮演、互动小说和世界观体验的创作者。你可以先搭建一个世界：写下背景、设计角色、准备开场、规划地点关系、配置模型与提示词，再让玩家进入这个世界，通过发言和行动推动剧情。

它更像一个“AI 叙事世界工作台”：

- 创作者负责世界观、角色、规则和表现方式。
- 玩家负责选择、行动、对话和探索。
- AI 负责根据上下文延展剧情、生成角色回应、维护世界状态。
- 系统负责保存、分支、调试、迁移和跨端呈现。

## 项目特点

### 🌍 世界包驱动

每个世界都可以作为独立内容单元存在，包含背景设定、开场剧情、角色池、地图拓扑、状态字段、提示词、桌面/移动 UI 和素材资源。世界可以被导出、导入、复用，也适合团队内部整理和交付。

### 🎭 多角色 AI 叙事

项目不是只围绕单一助手聊天，而是围绕“世界中的角色和事件”推进。角色可以拥有身份、长期提示、记忆策略、立绘资源和自定义信息；剧情中也可以根据上下文出现角色切换、旁白、系统事件和状态变化。

### 🧭 剧情存档与分支

玩家可以从已有存档继续，也可以从某个剧情节点创建分支，探索不同选择带来的路线变化。这让 AI 叙事更接近可回溯、可实验的互动故事，而不是一次性对话。

### 🧩 可定制游戏内 UI

世界包可以声明桌面端与移动端不同的游戏界面。消息列表、输入框、侧栏页签、浮动操作、场景头部、角色条、旁白卡等模块都可以通过运行时组件组合，让不同世界拥有更贴合自身题材的呈现方式。

### 📱 桌面与移动双端体验

项目支持 Windows 桌面应用和 Android APK。桌面端适合创作、配置和调试，移动端适合沉浸式游玩与随身体验。

### 🛠 调试链路完整

内置调试视图可以查看回合时间线、Prompt、记忆、错误、失败步骤和重试信息。对于需要反复打磨提示词、角色行为和剧情节奏的创作者来说，这部分非常关键。

### 🧠 本地数据与模型配置

世界、角色、会话、存档、模型配置和素材资源都围绕本地应用数据组织。项目支持配置文本模型、图像模型、默认模型、模型发现和连接测试，便于根据不同使用场景切换模型服务。

## 可以用来做什么

- AI 角色扮演游戏
- 互动小说和分支剧情体验
- 多角色世界观模拟
- 可迁移的 AI 剧情世界包
- 面向私有团队的叙事内容原型
- 带调试能力的 Prompt 和剧情规则实验平台

## 适合谁

- 想制作多角色、多场景、持续推进 AI 剧情世界的创作者。
- 希望把世界设定、角色资产、界面表现和运行规则一起打包的内容团队。
- 需要调试 Prompt、验证剧情逻辑、追踪 AI 回合链路的开发和测试团队。
- 想把 AI 对话体验做得更像“游戏”而不是“聊天框”的产品探索者。

## 当前状态

云朵梦境目前已经具备可运行应用、世界包、角色管理、存档分支、调试视图、模型配置、桌面/移动 UI 适配和跨端构建能力。项目仍在持续迭代，重点方向包括世界包 UI、移动端体验、叙事稳定性和内容创作流程。🚧
