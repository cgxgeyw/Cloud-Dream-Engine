# 云朵梦境

云朵梦境是一款以“世界”为核心单位的 AI 叙事游戏创作与体验工具。

你可以为一个世界配置角色、开场、地图、剧情提示、游戏内界面和素材资源，然后直接进入会话，让玩家在该世界中推进剧情、切换视角、查看状态，并在需要时保留存档与调试记录。

## 这个产品能做什么

- 创建和管理多个世界，每个世界都可以拥有独立的背景设定、开场内容、地图节点、剧情提示和状态页签。
- 为世界建立角色池，配置角色身份、长期提示、记忆策略、立绘资源和自定义页签内容。
- 启动游戏会话，让玩家在世界中发言、行动并推动剧情发展。
- 支持从已有存档继续游玩，也支持从任意节点分支出新的剧情路线。
- 提供调试视图，便于查看回合调用链、Prompt 记录、工具调用和异常回合信息。
- 支持将世界、角色、界面配置和依赖素材打包导出，再导入到另一台设备继续使用。
- 同时提供桌面端和 Android 端安装包，便于创作和体验在不同设备间切换。

## 适合谁

- 想要搭建多角色、多场景、可持续推进的 AI 剧情世界的创作者
- 需要按世界定制游戏内 UI 和素材表现的内容团队
- 需要保留调试链路、验证剧情推进逻辑的测试和内部使用团队

## 当前版本特点

- 以“世界包”为内容交付单位，适合整理、迁移和复用完整世界。
- 游戏内 UI 可以按桌面端和移动端分别配置。
- 角色、地图、状态面板和剧情推进都围绕同一个世界统一管理。
- 调试能力完整，适合内容迭代和问题排查。
